# Next Word Prediction Model (LSTM)

## 📖 Description
A deep learning-based NLP project designed to predict the next word in a sequence. By leveraging Long Short-Term Memory (LSTM) networks, the model learns the statistical structure and contextual dependencies within a given text dataset.

## 🧠 Key Features
* **Sequence Modeling:** Uses LSTM units to solve the vanishing gradient problem inherent in standard RNNs, allowing the model to learn long-range dependencies.
* **Text Preprocessing:** Includes custom pipelines for tokenization, sequence generation, and word embedding layers.
* **Deep Learning Framework:** Built entirely using TensorFlow/Keras for efficient GPU-accelerated training.

## 🛠 Tech Stack
* **Language:** Python
* **Deep Learning:** TensorFlow, Keras
* **NLP Libraries:** NLTK/Spacy (for text cleaning), NumPy

## 📈 Performance & Results
* accuracy: 0.9449
* Random dataset of a Data  Science course
